# Скрипт обработки CSV‑файлов для расчёта среднего рейтинга брендов
Назначение
Скрипт предназначен для:

- чтения данных о товарах из нескольких CSV‑файлов;

- расчёта среднего рейтинга для каждого бренда;

- вывода отчёта в виде таблицы в консоль.

Требования
- Python 3.8+

- Библиотека tabulate (для форматирования таблицы)
Устанавливается зависимости через файл requirements.txt
***

Пример запуска скрипта в командной строке:
python script.py --files products1.csv products2.csv --report average-rating
